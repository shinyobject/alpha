# AlphaGuess

Friends-only daily word game. Guess the hidden word — each guess tells you whether the target comes alphabetically before or after. Unlimited guesses.

## Stack

- Vite + React 18 + TypeScript
- Panda CSS
- Cloudflare Pages + Pages Functions (Hono)
- Workers KV
- Cloudflare Access (friend-group gating)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
